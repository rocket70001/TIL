#### RAID 구조  
Linear RAID == 순차 적재
RAID - 0 == 스트라이핑  
RAID - 1 == 미러링  
RAID - 5 == (N - 1, 하드 3개 이상) 스트라이핑 + 패리티 1   
RAID - 6 == (N - 2, 하드 4개 이상) 스트라이핑 + 패리티 2  
RAID - 0 + 1 == 스트라이핑 + 미러링  
RAID - 1 + 6 == 미러링 + 스트라이핑 + 패리티 2(아주 중요한 데이터)  
  
+ 쿼터 - 사용자가 사용할 수 있는 공간을 할당  

Telnet, SSH, XRDP 원격 통제
